# 📈 Multi-Platform Takeaway Sales: Data Pipeline & BI Dashboard

## 🚀 Project Overview
This repository tracks my hands-on progression from foundational Excel functions to advanced Business Intelligence architecture. The project culminates in an automated ETL data pipeline and dynamic dashboard that handles multi-platform delivery sales data (Uber Eats, Just Eat, Deliveroo).

---

## 📅 Learning & Development Timeline

### 🔹 Day 1: Foundational Analytics (`Day 1.xlsx`)
* Mastered core statistical aggregation functions: `SUM`, `AVERAGE`, `MIN`, and `MAX`.
* Implemented foundational logical testing with `COUNT` and basic conditional `IF` statements to segment sales thresholds.

### 🔹 Day 2: Advanced Logic & Aggregations (`takeaway_sales_analysis_D2.xlsx`)
* Expanded data architecture to handle multi-platform takeaway data streams.
* Engineered complex conditional logic using **Nested IF** structures to classify daily performance tiers.
* Used conditional counting (`COUNTIF`) to analyze volume distributions.

### 🔹 Day 4: Data Modeling & Visualization (`takeaway_sales_analysis_D4.xlsx`)
* Transitioned raw multi-platform tabular data into multidimensional **PivotTables**.
* Designed an interactive reporting view using integrated **PivotCharts** and synchronized **Slicers** for agile data filtering.

### 🔹 Day 6 (Cap Stone): The Automated ETL Pipeline (`takeaway_sales_analysis_D6.xlsx`)
* **Data Cleaning (Power Query):** Engineered an automated query to dynamically filter out manual summary row artifacts (`SUM`, `AVG`, `COUNT`), solve data noise, and normalize schema types.
* **Closed-Loop Dashboard:** Linked the final PivotTable structure directly to the cleaned Power Query output.
* **Result:** Replaced hours of repetitive weekly manual maintenance with a true, enterprise-grade **"Single-Click Refresh"** system (`Data -> Refresh All`).

---

## 🛠️ Tech Stack & Skills
* **ETL Engine:** Power Query / M Language
* **Analytics Layer:** Pivot Architecture, Logic Matrixing (`Nested IF`, `COUNTIF`)
* **Visualization:** Dashboard Design, Slicers, Clustered Column Charts
